Starting the game:

Mac Users:

Get Python 3.1:
  • I want to download it myself.
    Go to python.org, locate the Mac download and install it. You should now have an folder in your applications directory called "Python 3.1". Inside it there should be an application "Python Launcher.app". Go to "File > Open" in that application. Select the script "client.py" from wherever you saved it.

  • I don't want to fuss right now
    Along with the script "client.py" there should be a directory "python".
    Inside that there will be a directory "bin", and inside that there will be a file "python3.1".  Locate it.
    Now open the Application "Terminal".  It is located in "Applications > Utilities > Terminal".
    Drag the file "python3.1" onto the Terminal window.  Now drag the file "client.py" onto that same window.
    Now press Enter.
